#### 3.1 添加商品到购物车接口（POST /douyin/cart/add）
**测试状态：部分通过 ⚠️**
- 测试时间：2026-03-10
- 测试参数：
  ```json
  {
    "product_id": 1,
    "quantity": 1
  }
  ```
- 测试过程：
  1. 在商品列表页面点击iPhone 15 Pro的"加入购物车"按钮
  2. 查看提示信息和购物车状态
- 预期结果：返回200状态码，返回新增购物车项的id，购物车中成功添加该商品，数量为1
- 实际结果：
  - ✅ 接口调用成功，显示"已添加到购物车"提示信息
  - ❌ 购物车数量仍显示为0，购物车页面显示为空
  - 问题分析：前端添加到购物车成功后没有更新本地state.cart，导致购物车列表加载时显示为空
- 测试结果：⚠️ 部分通过（接口调用成功，前端状态更新有bug）
