## 四、发现的问题

### 4.1 已修复的问题

1. **apis 模块 go.mod 缺失** - 已创建
2. **配置文件 jaeger 选项错误** - 已改为 zipkin
3. **go.work 缺少 apis 目录** - 已添加
4. **前端代理静态文件路径错误** - 已修复
5. **前端 Mock 模式** - 已关闭

### 4.2 待修复的问题

#### 问题 1: 购物车列表返回数据不完整
- **现象**: 购物车 API 返回的商品缺少详情（名称、价格等）
- **原因**: 后端 carts 服务只返回购物车记录，未关联商品详情
- **影响**: 购物车页面显示商品名为 emoji，价格为 0
- **建议**: 修改 carts 服务，增加商品详情关联查询

#### 问题 2: order 服务未运行
- **现象**: 订单相关 API 无法使用
- **原因**:
  - RabbitMQ 用户认证失败 (username/password not allowed)
  - 缺少 order 服务的依赖配置
- **影响**: 无法创建订单、查看订单列表
- **建议**: 修复 RabbitMQ 配置或重新配置 order 服务

#### 问题 3: inventory 服务未运行
- **现象**: 库存服务相关 API 不可用
- **原因**:
  - MinIO 未运行（图片存储）
  - Elasticsearch 未运行（搜索功能）
- **影响**: 商品库存管理功能不可用
- **建议**: 启动 MinIO 和 Elasticsearch 容器

#### 问题 4: 优惠券和审计服务未启动
- **现象**: 优惠券和审计功能不可用
- **原因**: 未手动启动这些服务
- **建议**: 启动 coupons 和 audit 服务
